## About
Modern corporate website for Morph, featuring dynamic animations, video integration, and a responsive design.

## Features
- **Modern UI/UX**: Glassmorphism, text gradients, and smooth animations.
- **Dynamic Components**: Client logo carousel, team section, and video backgrounds.
- **Tech Stack**: React, Vite, Tailwind CSS, Framer Motion.

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```
